# OTT Clones - Cloudstream Extensions

## Install

Copy this URL and add it as a repository in Cloudstream:

```
https://raw.githubusercontent.com/sevouz/ottclones/builds/ottclones.json
```

## Plugins

| Plugin | Description | Language |
|--------|-------------|----------|
| netclone | Netflix, PrimeVideo, Disney+ Hotstar | Tamil |
| netmirrorclone | Netflix, PrimeVideo, Disney+ Hotstar | Hindi |
| movieboxinclone | Multi Language Movies and Series | Tamil |

| animedekhoclone | AnimeDekho, OnePace, HindiSubAnime | Hindi |
